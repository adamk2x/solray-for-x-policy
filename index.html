SolRay for X Privacy Policy

Effective Date: November 26, 2025

1. Data We Do Not Collect

The SolRay for X Chrome Extension is designed with user privacy as a top priority. We do not collect, store, transmit, or share any personal or sensitive user data.

Specifically, the extension does not collect:

  • Personally Identifiable Information (Name, Email, Address, etc.)
  • Browsing History
  • Authentication Information (Logins, Passwords)
  • User Activity (Keystrokes, mouse position)

2. Data Handling and Third-Party Services

The sole function of this extension is to retrieve public cryptocurrency token data.

  • Data Transmitted: When a user hovers over a Solana Contract Address (CA) on X/Twitter, the CA string (which is a public piece of data, not personal data) is sent from the extension's service worker to a third-party API.
  • Third-Party API: The extension makes API requests to https://api.dexscreener.com/ to fetch real-time token data (price, market cap, etc.).
  • Data Use: The token data received from the API is displayed only within the extension's hover card on the X/Twitter page and is not stored or transmitted by us.

3. Permissions Justification

The extension only uses the minimum necessary permissions:

  • activeTab: To interact with the content on the active X/Twitter page (e.g., to find the CA and display the hover card).
  • host_permissions for *://api.dexscreener.com/*: To securely fetch the public token data from the DexScreener API.

4. Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy here.
